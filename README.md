## How to Use

### Setup
1. Upload all three HTML files to your web hosting
2. Ensure all files are in the root directory or update paths accordingly
3. Update button links:
   - "Download App" → Link to App Store/Play Store
   - "Contact" section email links

### Customization
To modify app features or descriptions, edit:
- **index.html** - Line 244-295 (App features lists)
- **index.html** - Line 189-215 (App descriptions)

To update privacy policies, edit:
- **dkt-privacy.html** - Specific sections as needed
- **ace-privacy.html** - Specific sections as needed
